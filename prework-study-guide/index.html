HTML

  • The head element contain information about the webpage
  • The body element representes the visible content show to the user.
  • The style element defines the CSS styles associated with a website.
  • The base element defines the base URL for a webpage.
  • The link element connects an external resource to the HTML document.
  • The meta element defines metadta such as the character set, desciption, keywords, author, and viewport.
  • The scipt element embeds client-side scripts in the HTML document

CSS

  • A margin indicates how much space we want around the outside of an element
  • A padding indicates how much space we want around the content inside an element
  • To make the CSS formating appear in the HTML file you need to use the link element to creat a connection between the files
  • Class attributes allow to share a CSS rule to certain elelments, but not all elements

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git add -A: stages the working branch for changes
  • git commit -m "your message": creates a snapshot of the currently staged changes with a commit message
  • git pull origin main: gets the latest version of the main branch
  • git push origin feature/"branch": creates a new remote or branch or adds new changes to an existing one

JavaScript

  • A variable is a named container that allow us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • A function is a set of instruction that tells the computer how to perform a certain task.
  • An array is used to store a group of related data in a variable.
